# checkpoint02-Prompt-Toolkit--lce-

## Descrição
Neste projeto construimos um toolkit no python que automaticamente aplica técnicas de prompts a tarefas como negócios, comparação de resultados e recomendação de abordagem. 

## Tecnologias
- Python
- Pandas
- Matplotlib

## Como instalar e executar
Clone o repositório: git clone https://github.com/CaioSilva2026/checkpoint02-Prompt-Toolkit--lce-.git

Entre na pasta do projeto: cd checkpoint02-Prompt-Toolkit--lce-

Crie um ambiente virtual WINDOWS: python -m venv venv

Crie um ambiente virtual MAC/LINUX: python3 -m venv venv

Ativando o ambiente virtual WINDOWS: venv\Scripts\activate

Ativando o ambiente virtual MAC/LINUX: source venv/bin/activate

Instalando dependências: pip install -r requirements.txt

Executando o projeto: python main.py

Ou caso utilize o outro arquivo principal: python report.py
